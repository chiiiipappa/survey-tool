# サーベイBIツール

アンケート調査のレイアウトファイルと回答データをアップロードするだけで、クロス集計・グラフ作成・Excel エクスポートができるツールです。

---

## 概要

| 機能 | 内容 |
|---|---|
| STEP1 | レイアウトファイルを読み込み、集計軸・集計セットを設定する |
| STEP2 | 回答データ（CSV）をアップロードし、設問との照合・ラベル変換を行う |
| STEP3 | 集計軸ごとにクロス集計・グラフを生成し、Excel や PNG で出力する |

ブラウザ上で動作するローカルアプリです。インターネット接続は不要です。

---

## 動作環境

| 項目 | 要件 |
|---|---|
| OS | Windows 10 / 11、macOS 12 (Monterey) 以降 |
| Python | **3.10 以上**（3.11 推奨） |
| ブラウザ | Chrome / Edge / Safari（最新版） |

---

## インストール手順

### 1. Python をインストールする

Python がインストールされていない場合は、以下からダウンロードしてインストールしてください。

**https://www.python.org/downloads/**

> **Windows の方へ**：インストール時に「**Add Python to PATH**」にチェックを入れてください。

### 2. ZIP を解凍する

`サーベイBIツール.zip` を任意のフォルダに解凍してください。

---

## 起動方法

### Windows

`サーベイBIツール` フォルダを開き、

```
run.bat
```

をダブルクリックしてください。

- 黒いウィンドウ（コマンドプロンプト）が開きます
- 初回は数分かかる場合があります（ライブラリのインストール）
- 自動でブラウザが開き、ツールが表示されます

### Mac

**初回のみ**、ターミナルで以下のコマンドを実行してください（実行権限の付与）：

```bash
chmod +x /path/to/サーベイBIツール/run.command
```

その後、`run.command` をダブルクリック（またはダブルクリックして「開く」を選択）してください。

> **「開発元を確認できません」と表示された場合**：
> `run.command` を **右クリック** → **「開く」** を選択してください。
> 「開く」ボタンが表示されるので、クリックして起動してください。

## 使い方

### STEP 1 — レイアウト読み込み

1. 「レイアウトファイル」エリアに調査票レイアウト CSV（または Excel）をドラッグ＆ドロップ
2. 設問一覧が表示されます
3. 集計軸にしたい設問（性別・年代など）を選択
4. 「集計セット」を設定する

### STEP 2 — 回答データ読み込み

1. 「回答データ」エリアに回答 CSV をドラッグ＆ドロップ
2. 設問コードと列名の照合を確認
3. ラベル変換の結果を確認

### STEP 3 — クロス集計・グラフ

1. 集計軸を選択
2. 左サイドバーから集計セットを選択
3. グラフが自動生成されます
4. グラフ種別・色・ソートなどを変更可能
5. Excel または PNG でエクスポート

### プロジェクトの保存

右上の「保存」ボタンから `.surv` ファイルとして保存できます。次回起動時に読み込むと状態が復元されます。

> **注意**：ブラウザを閉じるか、サーバーを停止すると作業中のデータは消えます。こまめに保存してください。

---

## ツールの終了方法

起動時に開いた黒いウィンドウ（コマンドプロンプト / ターミナル）で `Ctrl + C` を押してください。

---

## トラブルシューティング

### 「ポートが使用中です」とエラーが出る

別のターミナルやプロセスがポート 8002 を使用しています。
以前の起動が残っている場合は、コマンドプロンプト / ターミナルを閉じてから再起動してください。

### 「Python が見つかりません」と表示される

Python がインストールされていないか、PATH に登録されていません。

- **Windows**：Python を再インストールし、「Add Python to PATH」にチェックを入れる
- **Mac**：ターミナルで `python3 --version` を実行して確認する

### Mac で「開けません」と表示される

`run.command` を **右クリック** → **「開く」** を選択してください。

### ブラウザが自動で開かない

ブラウザを手動で開き、アドレスバーに以下を入力してください：

```
http://localhost:8002
```

---

## 注意事項

- このツールはローカル（自分の PC）上でのみ動作します。外部からアクセスすることはできません
- アップロードしたデータはサーバーを停止すると自動削除されます
- インターネット接続は**不要**です（初回起動時のライブラリインストールを除く）
