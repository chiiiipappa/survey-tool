====================================================
 SurveyTool（サーベイBIツール）
====================================================

このツールでできること
----------------------------------------------------

・アンケートの集計作業を、ドラッグ＆ドロップだけで終わらせます
　（Excelで手作業で集計表を作る必要がありません）

・性別や年代など、知りたい軸ごとのクロス集計とグラフを
　自動で作ってくれます

・できあがった集計結果は、そのままExcelやPNG画像として
　保存・配布できます

・操作はすべて画面のボタンとドラッグ＆ドロップだけ。
　プログラミングや専門知識は一切必要ありません

・入力したデータは、お使いのパソコンの中だけで処理されます。
　インターネット経由で外部に送信されることはありません

「集計ツールなんて難しそう」と思われるかもしれませんが、
実際の操作はファイルを2つドラッグ＆ドロップするだけです。
このあとの手順どおりに進めれば迷うことはありません。


----------------------------------------------------
動作環境
----------------------------------------------------

・パソコン　　：Windows 10 / 11　または　Mac（macOS 12以降）
・インターネット：初回起動のときだけ必要です
　　　　　　　　　（2回目以降はインターネットがなくても使えます）


----------------------------------------------------
使い始める前に（最初の1回だけ）
----------------------------------------------------

このツールを動かすには「Python（パイソン）」という無料の
プログラムがパソコンに入っている必要があります。
難しいものではなく、インストールするだけで完了です。

すでにPythonを入れたことがある方は、この手順は不要です。
次の「インストール方法」に進んでください。

【手順】

  1. 以下のサイトをブラウザで開く

       https://www.python.org/downloads/

  2. 表示されている黄色いボタン（Download Python）を
     クリックしてダウンロードする

  3. ダウンロードしたファイルを開いてインストールする

     ★Windowsをお使いの方への注意★
     インストール画面の一番下にある
     「Add Python to PATH」という項目に、
     必ずチェックを入れてください。
     （このチェックを忘れると、あとでツールが正しく
       動かない場合があります）

  4. インストールが終わったら、いつものように
     パソコンを使える状態にしておく（再起動は基本不要です）

これで準備は完了です。


----------------------------------------------------
インストール方法
----------------------------------------------------

【Windowsをお使いの方】

  1. 「SurveyTool-Windows.zip」というファイルを
     右クリックして「すべて展開」を選ぶ

  2. 展開されたフォルダの中にある
     「SurveyTool Setup ○.○.○.exe」という
     ファイルをダブルクリックする

  3. 「Windows によって PC が保護されました」という
     画面が出ることがあります。これは無料配布のアプリで
     よく出る一般的な表示です。心配ありませんので、
     「詳細情報」という文字をクリックし、
     続けて「実行」ボタンを押してください

  4. 画面の案内に従って「次へ」「インストール」を
     クリックしていけば完了します


【Macをお使いの方】

  1. 「SurveyTool-Mac.zip」というファイルをダブルクリックして
     展開する

  2. 出てきた「SurveyTool-○.○.○.dmg」という
     ファイルをダブルクリックする

  3. 開いた画面の中の「SurveyTool」のアイコンを、
     隣にある「アプリケーション」フォルダの上まで
     マウスでドラッグする（これでインストール完了です）

  4. はじめてアプリを開くとき、
     「開発元を確認できないため開けません」と
     表示される場合があります。これは個人配布の
     アプリでよく出る一般的な表示です。
     その場合は、アイコンを「右クリック」して
     「開く」を選んでください
     （2回目以降は通常どおりダブルクリックで開けます）


----------------------------------------------------
起動方法
----------------------------------------------------

インストールした「SurveyTool」をいつものアプリと同じように
開いてください。

・はじめて起動したときだけ、画面の裏側で自動セットアップが
  行われます（数分かかります。画面に進行状況が表示されます）

・2回目からは数秒で起動します


----------------------------------------------------
使い方
----------------------------------------------------

[STEP 1] アンケートの設計図（レイアウトファイル）を読み込む

  1. 「レイアウトファイル」と書かれた枠に、調査票の
     レイアウトCSV（またはExcel）をドラッグ＆ドロップする
  2. 設問の一覧が表示されます
  3. クロス集計の軸にしたい設問（性別・年代など）を選ぶ
  4. 「集計セット」を設定する


[STEP 2] 回答データを読み込む

  1. 「回答データ」と書かれた枠に、回答のCSVファイルを
     ドラッグ＆ドロップする
  2. 設問と列が正しく対応しているか確認する
  3. ラベル変換の結果を確認する


[STEP 3] クロス集計とグラフを作る

  1. 集計したい軸を選ぶ
  2. 左側のメニューから集計セットを選ぶ
  3. グラフが自動的に作成されます
  4. グラフの種類・色・並び順なども変更できます
  5. Excel または PNG画像として保存できます


----------------------------------------------------
作業内容の保存について
----------------------------------------------------

画面右上の「保存」ボタンを押すと、作業内容を
ファイル（.surveyproject）として保存できます。

次にツールを開いたときにこのファイルを読み込めば、
続きから作業できます。

★注意★
ツールを終了すると、保存していない作業内容は消えます。
作業の途中でも、こまめに保存することをおすすめします。


----------------------------------------------------
終了方法
----------------------------------------------------

ウィンドウを閉じるだけで終了します。
（特別な操作は必要ありません）


----------------------------------------------------
困ったときは
----------------------------------------------------

●「Python が見つかりません」と表示される

  Pythonがインストールされていない、または
  「使い始める前に」の手順3でPATHのチェックを
  入れ忘れている可能性があります。
  もう一度「使い始める前に」の手順を確認してください。
  （Windowsの方は、Pythonを入れ直すときに
    「Add Python to PATH」へのチェックを忘れずに）


●初回起動時、セットアップが途中で失敗する

  インターネットに接続されているか確認してから、
  もう一度アプリを開き直してください。


●起動が遅い、または動かないことがある

  一度アプリを完全に終了し、少し時間をおいてから
  もう一度開いてみてください。


----------------------------------------------------
ご利用にあたっての注意
----------------------------------------------------

・このツールはお使いのパソコンの中だけで動作します。
　外部のサーバーやインターネット上にデータが
　送信されることはありません

・入力したデータは、アプリを終了すると自動的に
　消去されます。残しておきたい場合は
　「保存」ボタンで必ず保存してください

・インターネット接続が必要なのは、最初の1回だけです
